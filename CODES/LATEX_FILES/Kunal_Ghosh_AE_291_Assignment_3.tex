    or,

    Let, \[I = (n \times i) + j\]

    So, \[u^{k+1}_{I} = u^{k}_{I} + w(\hat{u}^{k}_{I} - u^{k}_{I})\]

    As,
\[\hat{u}^{k}_{I} = \frac{b_I - \displaystyle \sum^{n}_{J=I+1} a_{IJ}u^k_{J} - \displaystyle \sum^{I-1}_{J=1} a_{IJ}u^{k+1}_{J}}{a_{II}}\]
